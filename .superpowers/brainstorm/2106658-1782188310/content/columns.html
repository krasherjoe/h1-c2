スプレッドシート表示カラム設定

どの項目をテーブルに表示しますか？ 優先順にタップして選択してください。

現在の表示カラム（7列）

チェックを入れた項目がスプレッドシートに表示されます。

✓
商品名 (name)
商品の正式名称。一括編集で最も頻繁に変更される項目。
常時表示
✓
単価 (default_unit_price)
標準販売単価。税抜き／税込の表示切替対応。
常時表示
✓
バーコード (JAN) (barcode)
JANコード・EANコード。スキャナ連携、在庫管理で使用。
✓
型番 (model_number)
メーカー型番・管理番号。仕入発注時の識別に使用。
✓
カテゴリ (category / category_id)
所属カテゴリ。ドロップダウンで選択、ドラッグ&ドロップで変更。
✓
メーカー (manufacturer)
製造元・ブランド名。

追加可能カラム

以下の項目は現在非表示です。タップして表示カラムに追加できます。

○
卸値 (仕入価格) (wholesale_price)
仕入単価。粗利計算に使用。税抜き／税込切替対応。
○
在庫数 (stock_quantity)
現在庫数。在庫管理プラグインとの連携で自動更新。
○
仕入先 (supplier_name)
仕入先名。発注業務で使用。
○
商品名カナ (product_name_kana)
JAN標準対応。読み仮名での検索・ソートに使用。
○
メーカーコード (manufacturer_code)
JAN標準のメーカーコード。
○
分類コード (classification_code)
JAN標準の分類コード。業種別分類に使用。
○
説明 (description)
商品説明文。伝票・帳票への記載用。
○
タグ (tags)
任意タグ。グループ化・フィルタリングに使用。
○
有効期間 (valid_from / valid_to)
販売有効期間。期限切れ商品は自動非表示。

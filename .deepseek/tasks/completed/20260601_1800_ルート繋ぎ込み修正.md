# タスク: ダッシュボードのルート繋ぎ込み修正

**作成日時:** 2026-06-01 18:00  
**優先度:** P1（最優先）  
**担当:** OpenCode

---

## 目的

現在、ダッシュボードのボタンを押しても全て「準備中」のSnackBarが表示されるだけで、
実際の画面に遷移しない。各画面への正しいルーティングを繋ぎ込む。

---

## 問題の詳細

### 問題1: `dashboard_screen.dart` の `_navigate` がナビゲーションしていない

`lib/screens/dashboard_screen.dart` の141-145行目:

```dart
void _navigate(BuildContext context, MenuItem item) {
  ScaffoldMessenger.of(context).showSnackBar(
    SnackBar(content: Text('${item.title} - 準備中')),
  );
}
```

→ `Navigator.pushNamed(context, item.route)` に変更すること。

### 問題2: `main.dart` にルートが未登録

`lib/main.dart` の routes に以下が登録されていない:

- `/invoice/input` → `InvoiceInputForm`（`lib/screens/invoice_input/invoice_input_form.dart`）
- `/invoice/history` → `InvoiceHistoryScreen`（`lib/screens/invoice_history/invoice_history_screen.dart`）
- `/customer/master` → `CustomerMasterScreen`（`lib/screens/customer_master/customer_master_screen.dart`）
- `/product/master` → `ProductMasterScreen`（`lib/screens/product_master/product_master_screen.dart`）

---

## 要件

- [ ] `dashboard_screen.dart` の `_navigate` を `Navigator.pushNamed` に変更
- [ ] `main.dart` の `routes` に上記4画面を追加
- [ ] 各画面ファイルのimportを確認・追加
- [ ] ビルドエラーがないこと

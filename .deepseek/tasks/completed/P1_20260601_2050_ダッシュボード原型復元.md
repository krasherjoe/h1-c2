### Step 2: ダッシュボード書き換え

`lib/screens/dashboard_screen.dart` を以下の仕様で書き換える。

#### データソース

元は `AppSettingsRepository.getDashboardMenu()` で DB からメニューを取得していたが、
h-1-core ではプラグインレジストリからメニューを取得する。

```dart
// PluginRegistryはmain.dartで初期化済みのシングルトン
final registry = PluginRegistry.instance; // または適切な取得方法
final grouped = registry.getMenuItemsByCategory(); // Map<String, List<MenuItem>>
```

`MenuItem`（`lib/plugin_system/menu_item.dart`）は以下のフィールドを持つ:
- `id`, `title`, `route`, `category`, `icon`（IconData）, `description`

#### カテゴリ順序・説明

元の `kMenuCategoryOrder` / `kMenuCategoryDescriptions` と同じ内容を
`lib/screens/dashboard_screen.dart` 内の定数として定義する:

```dart
const List<String> _kCategoryOrder = [

const Map<String, String> _kCategoryDesc = {
```

#### ナビゲーション

元の `_buildTargetScreen` switch 文は不要。プラグインのルートは全て `main.dart` に登録済みなので:

```dart
void _navigate(BuildContext context, MenuItem item) {
  Navigator.pushNamed(context, item.route);
}
```

#### スライドロック

`SlideToUnlock` ウィジェットをそのまま使用。
ロック状態は `SharedPreferences` に保存（キー: `dashboard_history_unlocked`）。
ロック解除後のみ `/documents`（伝票管理）へ遷移可能。
ロック対象ルートは `/documents`。

```dart
void _navigate(BuildContext context, MenuItem item) {
  if (item.route == '/documents' && !_historyUnlocked) {
    ScaffoldMessenger.of(context).showSnackBar(
      const SnackBar(content: Text('スライドでロックを解除してください')),
    );
    return;
  }
  Navigator.pushNamed(context, item.route);
}
```

#### ステータスバー

`SharedPreferences` から取得（キー: `dashboard_status_enabled`, `dashboard_status_text`）。
デフォルト: enabled=true, text='販売アシスト1号 - 準備中'。

```dart
Widget _tile(MenuItem item) {
  return GestureDetector(
    onTap: () => _navigate(context, item),
    child: Container(
      margin: const EdgeInsets.only(bottom: 12),
      decoration: BoxDecoration(
        color: Theme.of(context).colorScheme.surface,
        borderRadius: BorderRadius.circular(12),
        boxShadow: [
          BoxShadow(
            color: Theme.of(context).colorScheme.onSurface.withValues(alpha: 0.1),
            blurRadius: 6,
            offset: const Offset(0, 2),
          ),
        ],
      ),
      padding: const EdgeInsets.all(16),
      child: Row(
        children: [
          CircleAvatar(
            radius: 22,
            backgroundColor: Theme.of(context).colorScheme.primaryContainer,
            foregroundColor: Theme.of(context).colorScheme.primary,
            child: Icon(item.icon),
          ),
          const SizedBox(width: 12),
          Expanded(
            child: Column(
              crossAxisAlignment: CrossAxisAlignment.start,
              children: [
                Text(item.title,
                  style: const TextStyle(fontWeight: FontWeight.bold, fontSize: 16)),
                const SizedBox(height: 4),
                Text('${item.id.toUpperCase()} • ${item.route}',
                  style: TextStyle(color: Theme.of(context).colorScheme.onSurfaceVariant, fontSize: 12)),
                if (item.description != null && item.description!.isNotEmpty) ...[
                  const SizedBox(height: 4),
                  Text(item.description!,
                    style: TextStyle(
                      color: Theme.of(context).colorScheme.onSurfaceVariant,
                      fontSize: 12)),
                ],
              ],
            ),
          ),
          Icon(Icons.chevron_right,
            color: Theme.of(context).colorScheme.onSurfaceVariant),
        ],
      ),
    ),
  );
}
```

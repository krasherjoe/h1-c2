## `CompanyPlugin` 仕様

```dart
class CompanyPlugin extends H1Plugin {
  @override String get id => 'com.h1.plugin.company';
  @override String get name => '自社情報';
  @override String get version => '1.0.0';
  @override String get description => '会社情報・印鑑・銀行口座の管理';

  @override List<MenuItem> getMenuItems() => [
    const MenuItem(
      id: 'CI',
      title: '自社情報',
      route: '/company',
      category: 'システム',
      icon: Icons.business,
      description: '会社名・住所・印鑑・口座',
    ),
  ];

  @override Map<String, WidgetBuilder> getRoutes() => {
    '/company': (_) => const CompanyProfileScreen(),
  };

  @override Future<void> initialize(PluginContext context) async {
    final repo = CompanyRepository(context.database);
    final profile = await repo.loadProfile();
    context.registerService<CompanyProfile>('companyProfile', profile);
  }

  @override Future<void> createTables(Database db) async {
    await db.execute('''
      CREATE TABLE IF NOT EXISTS company_info (
        id INTEGER PRIMARY KEY,
        name TEXT NOT NULL,
        zip_code TEXT,
        address TEXT,
        address2 TEXT,
        tel TEXT,
        fax TEXT,
        email TEXT,
        url TEXT,
        default_tax_rate REAL DEFAULT 0.10,
        seal_path TEXT,
        seal_offset_x REAL DEFAULT 10.0,
        seal_offset_y REAL DEFAULT 50.0,
        seal_rotation REAL DEFAULT 0.0,
        tax_display_mode TEXT DEFAULT 'normal',
        registration_number TEXT,
        bank_accounts TEXT,
        default_bank_account_index INTEGER DEFAULT 0,
        fiscal_year_start INTEGER DEFAULT 4,
        is_exempt_taxpayer INTEGER DEFAULT 0
      )
    ''');
  }
}
```

## `CompanyRepository` 仕様

現在の `SettingsRepository` のうちcompany_info DB操作部分を分離:

```dart
class CompanyRepository {
  final Database _db;
  CompanyRepository(this._db);

  Future<CompanyProfile?> loadProfile() async {
    final rows = await _db.query('company_info', limit: 1);
    if (rows.isEmpty) return null;
    return CompanyProfile.fromMap(rows.first);
  }

  Future<void> saveProfile(CompanyProfile profile) async {
    final existing = await _db.query('company_info', limit: 1);
    if (existing.isEmpty) {
      await _db.insert('company_info', profile.toMap());
    } else {
      await _db.update('company_info', profile.toMap(),
        where: 'id = ?', whereArgs: [1]);
    }
  }
}
```

## `SettingsPlugin` の変更点

- `company_info` テーブルの `createTables()` を**削除**（CompanyPluginが管理）
- `initialize()` での `CompanyProfile` サービス登録を**削除**（CompanyPluginが担当）
- `company_profile.dart` / `company_profile_screen.dart` のimportを削除
- `description` を `'印刷設定・伝票番号・税率'` に変更
- `dependencies` に `'com.h1.plugin.company'` を追加:
  ```dart
  @override List<String> get dependencies => ['com.h1.plugin.company'];
  ```

## 制約

- `company_info` テーブル定義は変更しない（DBマイグレーション不要）
- `CREATE TABLE IF NOT EXISTS` を使用（既存DBに影響なし）
- `SettingsPlugin` の `dependencies` に CompanyPlugin を追加すること
- null安全性・コメント最小限

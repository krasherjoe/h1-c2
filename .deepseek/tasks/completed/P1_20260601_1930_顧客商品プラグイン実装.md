## 実装仕様

### 顧客プラグイン

#### `customer_explorer_item.dart`
```dart
class CustomerExplorerItem extends H1ExplorerItem {
  final Customer customer;
  CustomerExplorerItem(this.customer);

  @override String get id => customer.id;
  @override String get title => customer.name;
  @override String? get subtitle => customer.phone; // 電話番号があれば
  @override String? get badge => null;
  @override IconData? get icon => Icons.person;
  @override DateTime? get updatedAt => customer.updatedAt;
}
```

#### `customer_explorer_config.dart`
```dart
class CustomerExplorerConfig extends H1ExplorerConfig<CustomerExplorerItem> {
  @override String get explorerTitle => '顧客マスター';
  @override String get searchHint => '顧客名・電話番号で検索';
  @override IconData get itemIcon => Icons.person;
  @override String get emptyMessage => '顧客が登録されていません';

  @override Future<List<CustomerExplorerItem>> fetchItems(String query) async {
    final repo = CustomerRepository();
    final customers = await repo.getAll(); // または検索付きメソッド
    if (query.isEmpty) return customers.map(CustomerExplorerItem.new).toList();
    return customers
        .where((c) => c.name.contains(query) || (c.phone?.contains(query) ?? false))
        .map(CustomerExplorerItem.new)
        .toList();
  }

  @override Widget buildViewer(BuildContext context, CustomerExplorerItem item) =>
    CustomerMasterScreen(selectionMode: false); // 既存画面を流用

  @override Widget buildEditor(BuildContext context, CustomerExplorerItem? item) =>
    CustomerEditScreen(customer: item?.customer); // 既存画面を流用

  @override Future<bool> canDelete(CustomerExplorerItem item) async => true;

  @override Future<void> deleteItem(CustomerExplorerItem item) async {
    await CustomerRepository().delete(item.customer.id);
  }
}
```

#### `customers_plugin.dart`
```dart
class CustomersPlugin extends H1Plugin {
  @override String get id => 'com.h1.plugin.customers';
  @override String get name => '顧客管理';
  @override String get version => '1.0.0';
  @override String get description => '顧客マスターの管理';

  @override List<MenuItem> getMenuItems() => [
    const MenuItem(
      id: 'C1',
      title: '顧客マスター',
      route: '/customers',
      category: 'マスター',
      icon: Icons.people,
      description: '得意先の登録・編集',
    ),
  ];

  @override Map<String, WidgetBuilder> getRoutes() => {
    '/customers': (_) => H1Explorer(config: CustomerExplorerConfig()),
  };

  @override Future<void> createTables(Database db) async {}
}
```

```dart
class ProductExplorerItem extends H1ExplorerItem {

  @override String get title => product.name;
  @override String? get subtitle => '¥${product.price}';
  @override String? get badge => product.category;
  @override IconData? get icon => Icons.inventory_2;
  @override DateTime? get updatedAt => product.updatedAt;
}
```

#### `product_explorer_config.dart`
```dart
class ProductExplorerConfig extends H1ExplorerConfig<ProductExplorerItem> {
  @override String get explorerTitle => '商品マスター';
  @override String get searchHint => '商品名で検索';
  @override IconData get itemIcon => Icons.inventory_2;
  @override String get emptyMessage => '商品が登録されていません';

  @override Future<List<ProductExplorerItem>> fetchItems(String query) async {
    final repo = ProductRepository();
    final products = await repo.getAll();
    if (query.isEmpty) return products.map(ProductExplorerItem.new).toList();
    return products
        .where((p) => p.name.contains(query))
        .map(ProductExplorerItem.new)
        .toList();
  }

  @override Widget buildViewer(BuildContext context, ProductExplorerItem item) =>
    ProductMasterScreen(selectionMode: false);

  @override Widget buildEditor(BuildContext context, ProductExplorerItem? item) =>
    ScreenP1ProductEditor(product: item?.product);

  @override Future<bool> canDelete(ProductExplorerItem item) async => true;

  @override Future<void> deleteItem(ProductExplorerItem item) async {
    await ProductRepository().delete(item.product.id);
  }
}
```

## 制約・注意事項

- `CustomerMasterScreen` と `ProductMasterScreen` の既存コードは**変更しない**
- H1ExplorerのConfigからラップして使う（既存コードの再利用）
- `CustomerRepository` / `ProductRepository` は既存のものを使用
- `CustomerModel` / `ProductModel` のフィールド名は実際のクラスを確認してから使うこと
- null安全性・mounted チェック必須
- コメントは最小限

## 目的

PDF出力・伝票発行に必要な会社情報・印刷設定を管理する設定プラグインを実装する。
他のプラグインが `PluginContext` 経由で会社情報を参照できるようにする。

## 実装仕様

### `company_profile.dart`

```dart
class CompanyProfile {
  final String name;         // 会社名
  final String postalCode;   // 郵便番号
  final String address;      // 住所
  final String tel;          // 電話番号
  final String fax;          // FAX
  final String email;        // メール
  final String? sealImagePath; // 印鑑画像パス
  final int defaultTaxRate;  // デフォルト税率 (10 or 8)
}
```

### `settings_repository.dart`

SharedPreferences に JSON で保存・読み込み:
- `saveCompanyProfile(CompanyProfile profile)`
- `Future<CompanyProfile> loadCompanyProfile()`

### `settings_screen.dart`

ListTileメニュー形式:
- 会社情報 → `company_profile_screen.dart`
- 税率設定（10% / 8% / 混在）
- 伝票番号の接頭辞設定

### `company_profile_screen.dart`

テキストフィールドで会社情報を入力・保存。

### `settings_plugin.dart`

```dart
class SettingsPlugin extends H1Plugin {
  @override String get id => 'com.h1.plugin.settings';
  @override String get name => '設定';
  @override String get version => '1.0.0';
  @override String get description => '会社情報・印刷設定';

  @override List<MenuItem> getMenuItems() => [
    const MenuItem(
      id: 'SET',
      title: '設定',
      route: '/settings',
      category: 'システム',
      icon: Icons.settings,
      description: '会社情報・印刷設定',
    ),
  ];

  @override Map<String, WidgetBuilder> getRoutes() => {
    '/settings': (_) => const SettingsScreen(),
  };

  @override Future<void> createTables(Database db) async {}

  @override Future<void> initialize(PluginContext context) async {
    final repo = SettingsRepository(context.preferences);
    final profile = await repo.loadCompanyProfile();
    context.registerService<CompanyProfile>('companyProfile', profile);
  }
}
```

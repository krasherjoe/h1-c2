## 実装仕様

### `h1_explorer_item.dart`

```dart
abstract class H1ExplorerItem {
  String get id;
  String get title;
  String? get subtitle;
  String? get badge;       // 例: "請求" "顧客" などのラベル
  IconData? get icon;
  DateTime? get updatedAt;
}
```

### `h1_explorer_config.dart`

```dart
abstract class H1ExplorerConfig<T extends H1ExplorerItem> {
  String get explorerTitle;          // AppBarに表示するタイトル
  String get searchHint;            // 検索フィールドのヒント
  IconData get itemIcon;            // デフォルトアイコン
  String get emptyMessage;          // 空のとき表示するメッセージ

  Future<List<T>> fetchItems(String query);
  Widget buildViewer(BuildContext context, T item);
  Widget buildEditor(BuildContext context, T? item); // null=新規
  Future<bool> canDelete(T item);   // 削除可能か判定
  Future<void> deleteItem(T item);
}
```

### `h1_explorer.dart`

以下の機能を持つ `StatefulWidget`:

**一覧モード（初期表示）**
- `AppBar`: タイトル + 検索アイコン + 新規作成FAB
- 検索バー: テキスト入力で `fetchItems(query)` を呼ぶ
- `ListView`: 各行は `ListTile` ベース（icon, title, subtitle, badge）
- 行タップ → ビューワへ遷移（`Navigator.push`）
- 行の長押し or スワイプ → 削除確認ダイアログ（`canDelete`がtrueのみ）
- Pull-to-refresh対応

**ビューワモード**
- `buildViewer(context, item)` の結果を表示
- AppBarに「編集」ボタン → エディタへ遷移

**エディタモード**
- `buildEditor(context, item)` の結果を表示（item=nullなら新規）
- AppBarに「保存」「キャンセル」

**遷移フロー**
```
一覧
  ├─ 行タップ → ビューワ（push）
  │     └─ 編集ボタン → エディタ（push）
  └─ FAB → エディタ（新規, push）
```

## 制約・注意事項

- `H1Explorer` 自体はデータを一切持たない（全てConfigに委譲）
- Flutter標準ウィジェットのみ使用（追加パッケージ不要）
- null安全性を遵守
- mounted チェック必須
- コメントは最小限
- このStep 1ではConfigの**具体実装は作らない**（骨格のみ）
- ビルドが通ることを確認すること

## 実装仕様

### 売掛一覧 (`accounts_receivable_screen.dart`)

- 請求済み・未入金の伝票一覧（`documents` テーブルを参照）
- 顧客別集計・合計金額表示
- 入金登録ボタン

### 支払スケジュール (`payment_schedule_screen.dart`)

- 月別の支払予定カレンダー/リスト表示
- 仕入 (`purchases` テーブル) の未払い集計

### 入金登録 (`payment_register_screen.dart`)

- 顧客・金額・日付・備考
- 対応する請求伝票に紐付け

### 資金繰り表 (`cash_flow_screen.dart`)

- 月別の入金（売上）vs 出金（仕入・費用）の集計表示
- `documents` + `purchases` テーブルを集計

### DBテーブル

```sql
CREATE TABLE IF NOT EXISTS payments (
  id TEXT PRIMARY KEY,
  type TEXT NOT NULL,        -- 'received' | 'paid'
  customer_id TEXT,
  supplier_id TEXT,
  document_id TEXT,
  amount INTEGER NOT NULL,
  date TEXT NOT NULL,
  note TEXT,
  created_at TEXT
);
```

## 実装仕様

### `stock_transaction_model.dart`

```dart
enum StockTransactionType {
  inbound,    // 入庫
  outbound,   // 出庫
  transfer,   // 移動
  adjustment, // 調整
  stocktake,  // 棚卸
}

class StockTransaction {
  final String id;
  final StockTransactionType type;
  final String productId;
  final String productName;
  final double quantity;   // 正=入庫、負=出庫
  final DateTime date;
  final String? note;
}
```

### `inventory_list_screen.dart`（在庫一覧）

- 商品ごとの現在庫数を表示（`stock_transactions` を集計）
- 検索・ソート対応
- 各行タップで入出庫履歴

### 入出庫画面（`stock_inbound_screen.dart`, `stock_outbound_screen.dart`）

- 商品選択（`ProductRepository`）
- 数量入力
- 日付・メモ
- 保存で `stock_transactions` に記録

### DBテーブル

```sql
CREATE TABLE IF NOT EXISTS stock_transactions (
  id TEXT PRIMARY KEY,
  transaction_type TEXT NOT NULL,
  product_id TEXT NOT NULL,
  product_name TEXT NOT NULL,
  quantity REAL NOT NULL,
  date TEXT NOT NULL,
  note TEXT,
  created_at TEXT
);
```

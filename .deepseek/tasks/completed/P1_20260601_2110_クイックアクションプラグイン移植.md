## 目的

元の `h-1.flutter.0` にあった「クイックアクション」機能を
`h-1-core` にプラグインとして移植する。

**機能概要:**
- アイコングリッドで機能を素早く呼び出せるホーム画面風UI
- 複数ページ（PageView）を左右スワイプで切り替え
- ページ・アクションを自由にカスタマイズ可能
- 設定画面でページ追加・並び替え・アクション割り当て

## 実装仕様

### `quick_action_page.dart`

```dart
class QuickActionPage {
  final String id;
  String name;
  int sortOrder;
  List<String> actionIds; // ルート文字列のリスト

  QuickActionPage({required this.id, required this.name,
    this.sortOrder = 0, List<String>? actionIds})
      : actionIds = actionIds ?? [];

  Map<String, dynamic> toJson() => {'id': id, 'name': name,
    'sortOrder': sortOrder, 'actionIds': actionIds};

  factory QuickActionPage.fromJson(Map<String, dynamic> json) =>
    QuickActionPage(
      id: json['id'] as String? ?? '',
      name: json['name'] as String? ?? '新規ページ',
      sortOrder: json['sortOrder'] as int? ?? 0,
      actionIds: (json['actionIds'] as List?)?.cast<String>() ?? [],
    );
}
```

**アイコングリッドのレイアウト:**

**アクション定義:**
PluginRegistry から全メニュー項目を取得してルート→MenuItem のマップを作る:

```dart
Map<String, MenuItem> get _allActions {
  final registry = PluginRegistry.instance;
  final map = <String, MenuItem>{};
  for (final item in registry.getAllMenuItems()) {
    map[item.route] = item;
  }
  return map;
}
```

### `quick_actions_plugin.dart`

```dart
class QuickActionsPlugin extends H1Plugin {
  @override String get id => 'com.h1.plugin.quick_actions';
  @override String get name => 'クイックアクション';
  @override String get version => '1.0.0';
  @override String get description => 'ホーム画面風のショートカット';

  @override List<MenuItem> getMenuItems() => [
    const MenuItem(
      id: 'QA', title: 'クイックアクション',
      route: '/quick_actions',
      category: 'システム',
      icon: Icons.grid_view,
      description: 'ショートカットメニュー',
    ),
  ];

  @override Map<String, WidgetBuilder> getRoutes() => {
    '/quick_actions': (_) => const QuickActionsScreen(),
    '/quick_actions/settings': (_) => const QuickActionSettingsScreen(),
  };

  @override Future<void> createTables(Database db) async {}
}
```

List<MenuItem> getAllMenuItems() {

    .expand((p) => p.getMenuItems())
    .toList();
}
